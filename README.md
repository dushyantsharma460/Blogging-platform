## ⚡ Features

- JWT Authentication
- Create, Edit & Publish Blogs
- Serverless Backend using Hono
- Cloudflare Workers Deployment
- Prisma ORM
- Prisma Accelerate Connection Pooling
- Neon PostgreSQL Database
- Fully Responsive UI
- Fast Edge API Performance

---

## 🛠 Deployment

Backend deployed using:

- Wrangler
- Cloudflare Workers

Database:

- Neon PostgreSQL
- Prisma Accelerate
